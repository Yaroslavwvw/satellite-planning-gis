# Frontend (React + Vite + TypeScript)

## Запуск

```bash
cd frontend
npm install
npm run dev
```

## Переменные окружения

- `VITE_API_BASE_URL` (опционально) — URL backend API, по умолчанию `http://localhost:8000`

## Что уже есть

- Русскоязычный каркас интерфейса по макету: верхний хедер, левая панель параметров, карта Leaflet, нижняя панель результатов.
- Страницы: Расчет, Спутники, О системе, Сохраненный расчет.
- Базовые API-клиенты в `src/api`.
